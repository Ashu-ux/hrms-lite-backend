## Deployment (Render)

1. Push backend repo to GitHub
2. Create Web Service on Render
3. Add environment variable:


DATABASE_URL = [https://hrms-lite-backend-czi9.onrender.com]


4. Build command:


npm install && npx prisma generate
```

5. Start command:


node server.js
```
